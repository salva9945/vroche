# Vroche Landing Page

Landing page responsive para Vroche, preparada como proyecto React + Vite + Tailwind.

## Cómo ejecutarla en local

```bash
npm install
npm run dev
```

## Cómo generar build de producción

```bash
npm run build
```

El build se generará en la carpeta `dist/`.

## Deploy rápido

Puedes subir este proyecto directamente a Vercel, Netlify o cualquier hosting compatible con Vite.
