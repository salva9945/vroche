# Vroche Landing Spectacular

Landing page responsive para Vroche con:

- Hero fashion-tech.
- Logo real de Vroche.
- Fondo interactivo con logos pequeños sin deformar.
- Scroll storytelling con modelo que se viste por capas.
- Secciones de producto, experiencia, visión y waitlist.
- Responsive mobile/tablet/desktop.

## Ejecutar en local

```bash
npm install
npm run dev
```

## Build producción

```bash
npm run build
```
